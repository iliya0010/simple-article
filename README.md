#### Simple blog 

This is a simple blog for a test.


### Requirements

To run this code you need to have following tools:

- docker
- docker-compose

### How to install

Just run following commands in the root of the app to start the project :

``` docker-compose build ```

and then :

``` docker-compose up -d ```
